Operadores e Lógica

Os operadores são os motores do seu código. Eles pegam as variáveis (os dados) e realizam transformações neles.

Operadores Aritméticos

  • +, -, *, /: As quatro operações básicas.
  • % (Módulo): Retorna o resto da divisão. Essencial para descobrir se um número é par ou ímpar (x % 2 == 0).

Incremento e Decremento

int x = 10;
x++; // vira 11
x--; // volta para 10

Comparações e Decisões Lógicas

Aqui é onde seu programa começa a "pensar".

Relacionais (O resultado é sempre boolean)

  • ==: Igual a.
  • != Diferente de.
  • >, <, >=, <=: Comparações de magnitude.

Lógicos (Combinando condições)

  • && (E): Retorna true só se AMBOS forem true.
  • || (OU): Retorna true se PELO MENOS UM for true.
  • ! (NÃO): Inverte o valor (true vira false).

Precedência: Quem manda em quem?

Assim como na matemática, o Java resolve as coisas em uma ordem específica. Se você não cuidar, o resultado pode sair errado.

Bitwise: Lógica no Nível dos Átomos

Os operadores bitwise manipulam os bits (0 e 1) individuais dos números. Isso é usado em criptografia, compressão e sistemas de alta performance.

  • & (Bitwise AND).
  • | (Bitwise OR).
  • ^ (Bitwise XOR).
  • ~ (Bitwise NOT).
  • << (Left Shift): Desloca os bits para a esquerda, multiplicando o número por 2 a cada passo.
